Detexify : LaTeX 's symbol finder

Anyone who works with LaTeX knows how time-consuming it can be to find a symbol in symbols-a4.pdf that you just can't memorize. Detexify is an attempt to simplify this search. Visit http://detexify.kirelabs.org/classify.html

Vector class using Expression Templates

The Template Metaprogramming in C++ allows us to use templates to represent expressions. It also allows to perform lazy evaluations of the expressions which reduces the allocation of temporary variables in calculations, hence increasing the performance of the code. More info on this can be found on Wikipedia or other sources. Here's a small example which I wrote to represent a vector of type double. The code uses Curiously Recurring Template Pattern.
