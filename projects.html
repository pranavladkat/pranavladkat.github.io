3D Unsteady Vortex Panel Method

This project is still in active development. I'm developing a general purpose 3D Unsteady Vortex Panel method as a part of my Master's Project. Vortex Panel Method is widely used in the early design of streamlined bodies such as airfoils. The method gives fairly accurate results of pressure, drag and lift forces and moments on the body. The motivation of the present work is to take advantage of fast solution time of the this method and use it in real time control feedback system of a wind turbine.

The method is constant strength Source-Doublet Panel Method. The present code is developed in a object oriented C++ making use of modern C++ features.

2D Plane Stress Analysis using Finite Element Method

I developed this code to solve 2D plane stress problem as a part of course project for "Finite Element Methods" in Fall 2014. The code is written in C++ making use of its OOP concepts.

The Plane Stress is a simplification of the elastostatic problem when the stress vector is zero across a particular surface. A bilinear quadrilateral element is implemented and 2 point Gaussian Quadrature is used to solve the integration numerically. The code can import geometry in an unstructured mesh file format which defines geometry and boundary conditions. The current code is validated with ANSYS APDL commercial package and the solution is matched within tolerance of 1E-4 units.

Simulating Two-Phase Incompressible Flow using Level Set Method

I developed a two-phase flow solver as a part of course project for "Computational Moving Interfaces" during summer 2014. The code is written in C making use of PETSc library.

The code utilizes second order Projection Method with variable density and viscosity to solve the incompressible flow field. A collocated grid is opted for the current work. The interface between two phases is sharp and tracked with the zero level set of smooth function. The projection step is solved using GMRES solver. The Godunov method is used for solving reinitialization and level set advection equations. Uses adaptive time-stepping to advance in time. Continuum Surface Force is used for modeling Surface Tension.

Parallel SOR using MPI, OpenMP and CUDA

The program solves a generic Poisson Equation using Red-Black Successive Over Relaxation method. The aim of the program is to take advantage of multiple processors using MPI, OpenMP and CUDA. This project is done for a "High Performance Computing" course in Fall 2014.

The code is written in object oriented C++ to mimic a behavior of a numerical library. Code lets user set up a Poisson Equation such as coefficients and right hand side function, Grid parameters, Boundary Conditions via user defined functions, solver parameters such as solver type, tolerance etc. and then solves it. The MPI and OpenMP solver achieves a nearly linear speedup when number of processors are increased. The CUDA code achieves speedup up to ~50 compared to 1 CPU and ~5 compared to 16 processors.

Incompressible Navier-Stokes flow solver in Curvilinear Coordinates using C++

I developed a Navier Stokes Flow solver in Curvilinear Coordinates as a course project for "CFD-II" in spring 2014. The code uses Artificial Compressibility Method for pressure-velocity coupling. The code is second order accurate in space and fourth order accurate in time due to use of 4th order Runge-Kutta. The code can be used with stretched grids and curved grids. The code is validated with examples of Cavity flow and Backward facing step problems.

Numerical Methods for Capturing Shocks

The hyperbolic equations with the shocks in its initial condition are solved with some of the conservative schemes such as Godunov Method, Godunov Method with MUSCL scheme and Modified Runge-Kutta with TVD. The classical problem of Dam-Break problem (1D) is simulated. The code is implemented in MATLAB. The project was a part of CFD-I course.

Solution to Elliptical PDE's using SOR and ADI

The Elliptic PDE's in 2D are solved using iterative methods such as Successive Over Relaxation and Alternating Direction Implicit methods. Used MATLAB for implementing these methods. The project was a part of CFD-I course.
